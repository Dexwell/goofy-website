Introducing Goofy

Your desktop client for Facebook Messenger

Never lose your chat window among all the other open browser windows and get notifed about new messages.

Notifications

Don't miss any message from your friends. Goofy uses the built-in system notifications and Dock badge to inform you about new messages.

Fully featured

Goofy uses Facebook's very own web-chat and therefore has all its features. Including stickers, group chats, search and more...

Free

I wrote this App because it helps me to better use Facebook Messenger. So why shouldn't you have it, too?

Nice,...but how?

Basically this is just a web-view of facebook.com/messages. I just add some JavaScript and CSS to make it fancy. Learn more!
